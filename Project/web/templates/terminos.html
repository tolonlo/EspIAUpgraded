Términos y Condiciones

Este sitio web es un proyecto universitario que permite controlar la reproducción de música mediante gestos utilizando la cámara del dispositivo.

  • El usuario debe permitir el acceso a la cámara para que la detección de gestos funcione.
  • La música reproducida es solo para uso personal y educativo, no debe ser distribuida ni utilizada con fines comerciales.
  • La cámara se usa únicamente para la detección de gestos y no se almacenan videos ni imágenes.
  • No se recopila ni comparte información personal del usuario.
  • No nos hacemos responsables por daños en dispositivos o pérdida de datos.
  • El usuario usa esta aplicación bajo su propio riesgo.
  • Los términos pueden cambiar sin previo aviso.
  • El usuario debe revisar periódicamente estos términos.

Para dudas o sugerencias, contacte al equipo de desarrollo del proyecto universitario.
